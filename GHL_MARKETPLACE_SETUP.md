## 1. App-Erstellung (Initial Setup)

### Pflichtfelder beim Erstellen:

#### App Name *
- **Empfehlung:** "Website Cloner" oder "Universal Website Importer"

#### App Type
- **Auswahl:** Private oder Public
- **Empfehlung:** Private (für Testing), später Public

#### Target User *
- **Auswahl:** Sub-Account [Recommended] oder Agency
- **Empfehlung:** Sub-Account (Recommended)
- **Begründung:** Die App wird von Sub-Accounts verwendet, um Websites zu importieren

#### Who can install the app?
- **Auswahl:** Everyone oder Agency Only
- **Empfehlung:** Everyone
- **Begründung:** Maximale Reichweite

#### Listing Type
- **Auswahl:** White-label (Recommended) oder Standard (Non white-label)
- **Empfehlung:** White-label (Recommended)
- **Begründung:** Ermöglicht Agenturen, die App unter eigenem Branding anzubieten

## Technische Anforderungen

### OAuth 2.0 Integration
- **Client ID** und **Client Secret** von GHL
- **Redirect URI** konfigurieren
- **Scopes** definieren:
  - `locations.readonly`
  - `locations.write`
  - `funnels.readonly`
  - `funnels.write` (falls verfügbar)
  - `medias.write`

### Webhook-Endpoints
- Installation-Webhook
- Uninstallation-Webhook
- Update-Webhook

### SSO (Single Sign-On)
- GHL SSO-Integration für nahtlosen Login
